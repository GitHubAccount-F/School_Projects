# Demonstrates we can book multiple Itineraries at once

# commands:
create user1 user1 100000
login user1 user1
search "Seattle WA" "San Francisco CA" 1 5 1
book 0
search "Seattle WA" "San Francisco CA" 1 4 1
book 0
quit
*

# expected command output:

Logged in as user1
Itinerary 0: 1 flight(s), 112 minutes
ID: 1028732 Day: 5 Carrier: VX Number: 755 Origin: Seattle WA Dest: San Francisco CA Duration: 112 Capacity: 16 Price: 674

Itinerary 0: 1 flight(s), 110 minutes
ID: 464739 Day: 4 Carrier: UA Number: 156 Origin: Seattle WA Dest: San Francisco CA Duration: 110 Capacity: 12 Price: 652
Booked flight(s), reservation ID: 2
Goodbye
